# MC Resonance Horizons
The Official Repo for MC Resonance Horizons 1.21.1 modpack, put together by ishinyca. Contains many tech, adventure, and magic mods. Something for everyone! Hybrid pack with some fun and cool challenges to it.

Important info: This modpack will change and may break with big updates. So I'm tagging it as beta (B). I'll try my best to not break every update ever. Also, expect major balance changes until the official 1.0 release.

If there are any bugs that you come across and is related to the modpack itself, please report it in the issues. I will take a look at them to see if it is modpack related and not a mod related.

Note: Originally called ishinyca's MC Adventure but after many ideas of coming up with a name, I finally setlled on the "MC Resonance Horizons" name.
